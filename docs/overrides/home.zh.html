让MkDocs本地化变得更简单

通过 mkdocs-static-i18n 插件, 您可以在现有文档页面中添加静态翻译文件，从而支持多语言文档。

MkDocs static i18n插件尽可能地简单易用，它会生成网站的默认语言版本 + 其他语言的版本，每个版本对应一种配置语言，对应语言的版本将放置在<language>/路径下。

更棒的是，mkdocs-static-i18n还允许您构建和提供任何文件的本地化版本，以显示多语言版本的图片、媒体和资源，并支持自动回退到默认语言的页面和资源。

注重于翻译本身,而不是花费太多精力和时间去更新与每种本地化语言相关的资源和链接。
